Basics of Object Oriented Programming and Java Syntax

In java to make a comment you would go to the link and add two forward slashes (//) in front of the line of the code that you have written.

A comment is used for when you are explaining your code. Explaining your code is a crucial part of programming. Many people are unsure of how your code works especially if it is someone's first time reading code or doing the code. This is for more clarification with your code whether that is someone who does not know much about coding or someone who has been coding for a while. It is equally as important to make sure that your program is filled with information to tell you about your code.

As talked about previously, having your code commented well throughout can help someone understand it better, in Java this is called JavaDocs. JavaDocs are able to be produced by using the /** for the beginning and then **/ at the end indicating that the JavaDocs have started and ended.
